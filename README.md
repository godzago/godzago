## 🤖 AI-Assisted Development & Vibe Coding

I actively use modern AI tools as part of my development workflow.

For me, AI-assisted development is not only about generating code. It is about moving faster from **idea → prototype → testable product**, while still understanding the architecture, business logic, data flow, and long-term maintainability behind the software.

I use AI tools for:

- Exploring product ideas
- Creating first versions of screens and flows
- Generating boilerplate code
- Refactoring and simplifying code
- Reviewing architecture decisions
- Writing documentation
- Testing API flows
- Building small developer tools

In 2026, I see AI-assisted development and vibe coding as practical parts of modern software development, especially for MVPs, internal tools, creative software projects, and fast product experiments.
